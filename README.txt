
Functionality extensions:

Logic:
Up to 4 players can play the game.
Multiple human players can play the game (this has to be set in the code).
Text is sized depending on the dimensions of the grid displayed.

Useer interface:
Before a game is played, the user is presented with a screen that allows them to
set the grid size, up to 6 by 6 squares. Any larger would make it difficult to
touch an individual line.
